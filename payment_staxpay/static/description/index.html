Overview

Add Staxpay as payment method in your website

Usage

  1. After installation, navigate to Website > Configuration > Payment providers. Click on Staxpay and fill the "Staxpay public api key" field that you get from your merchant Staxpay account.
  2. Set the sandbox API key to do test payments in test mode, or enter the production key and enable the payment acquirer

  3. Add Staxpay payment method in Journal Account set for the provider
